# widgets.systematicyou.ai

Embeddable widgets for Systematic You dashboards (Notion, web).

## Live widgets

- `/pomodoro/` — Focus Timer (Pomodoro + Custom mode, forced 5-min break, skip-break)

## Embed in Notion

In any Notion page, type `/embed`, paste the widget URL (e.g. `https://widgets.systematicyou.ai/pomodoro/`), and resize the embed block to taste.

## Brand

All widgets follow the Max Design System brand palette (Iron Gate, Mahogany Desk, Dusty Rose, Blush Hour, Biscotti) with EB Garamond for headings/body and IBM Plex Mono for tabular figures.
